XNJB created by Richard Low <richard@wentnet.com>.

Based on libnjb and libmtp.

Uses libusb (http://libusb.sourceforge.net/), ID3Tag.framework (http://drewfamily.homemail.com.au/Cocoa_-_ID3Tag_framework.html) and some code based on Gnomad2 (http://gnomad2.sourceforge.net/). Please see enclosed licence files for information on copying these libraries.
